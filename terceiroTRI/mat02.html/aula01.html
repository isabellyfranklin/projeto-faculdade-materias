Sistema de Gerenciamento de Banco de Dados (SGBD)

O que é um SGBD

Um SGBD (Sistema de Gerenciamento de Banco de Dados) é um software usado para armazenar, manter e acessar grandes volumes de dados de forma eficiente e segura.

  • Manutenção: inserir, alterar e excluir dados.
  • Utilização: buscar e ler dados rapidamente.

Exemplo: se um sistema precisa guardar informações de candidatos da Justiça Eleitoral, o SGBD garante que os dados fiquem salvos mesmo após desligar o computador e que possam ser recuperados com rapidez e segurança.

Por que usar um SGBD

Usar um SGBD traz várias vantagens:

  • Facilita o acesso aos dados sem precisar saber como eles estão armazenados.
  • Possui algoritmos otimizados para buscas rápidas.
  • Garante segurança e integridade dos dados.
  • Permite acesso simultâneo** de vários usuários.
  • Ajuda na recuperação de falhas e reduz o tempo de desenvolvimento.

Exemplos de SGBDs: MySQL, PostgreSQL, Oracle, SQLite, SQL Server, MongoDB.

Dado, informação e conhecimento

  • Dado: é algo bruto, como números, textos ou pixels de uma imagem.
  • Informação: é o significado* que o ser humano dá a esses dados (exemplo: ver uma foto e entender quem ganhou um prêmio).
  • Conhecimento: é o resultado da interpretação e experiência pessoal com as informações.

O livro reforça que dados pertencem ao computador, mas informações e conhecimento pertencem aos humanos — e que só praticando (criando e manipulando bancos) é possível realmente aprender.

Bela, registro e campo

Em um banco de dados relacional:

  • Os dados são organizados em tabelas (como planilhas).
  • Cada linha é um registro (uma ocorrência de algo, como um funcionário).
  • Cada coluna é um campo (um atributo, como nome, salário, etc.).

Exemplo: uma tabela “municípios” tem as colunas nome, uf, população, área e cada linha representa uma cidade.

Boas práticas

  • Não usar acentos em nomes de tabelas e colunas.
  • Usar letras minúsculas e "underscore" (`_`) para separar palavras

Consideração final

O SGBD é essencial no desenvolvimento de sistemas porque organiza, protege e facilita o acesso aos dados de forma profissional.

Ele substitui o trabalho manual e difícil de armazenar informações em arquivos, oferecendo segurança, velocidade e praticidade.

Além disso, compreender a diferença entre dado, informação e conhecimento mostra que a tecnologia só tem valor quando o ser humano interpreta e usa os dados com propósito.

  • Em resumo:o banco guarda os dados, mas é o programador quem transforma esses dados em informação e conhecimento
